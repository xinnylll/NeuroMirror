# NeuroMirror: A Cybernetic Journey Through Emotion

## Tagline
An embodied web-based interactive installation that translates real-time facial expressions into dynamic generative landscapes and physiological feedback.

## Project Description
NeuroMirror is a novel human-computer interaction (HCI) experiment exploring the intersection of computer vision, generative art, and behavioral intervention. Utilizing MediaPipe's Face Landmarker, the system captures real-time 3D facial blendshapes and processes them through a custom heuristic state machine based on Paul Ekman's Facial Action Coding System (FACS). 

## Human-Centered Design Analysis

### Affordances & Signifiers
* **Affordances:** The real-time camera feed afford users the action of performing emotions.
* **Signifiers:** The progress bars (Happy, Sad, Angry) provide immediate visual cues of the system's detection.

### Feedback Loops (Cybernetic System)
The project functions as a dynamic feedback system:
1. **Input:** User's facial micro-expressions.
2. **Processing:** Heuristic algorithms calculate emotional intensity and neurotransmitter states.
3. **Output:** Generative visual art (Rollercoaster, Rain, or Volcano) and psychological suggestions.
4. **Behavioral Influence:** As users observe their "Adrenaline Spike" or "Dopamine Surge," the system suggests actions (e.g., Deep Breathing), prompting the user to regulate their biological state.
